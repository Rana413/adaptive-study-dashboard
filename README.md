# AI-Based Adaptive Study Dashboard

## Project Overview
This project demonstrates a simple AI-based adaptive user interface system. The dashboard adapts its behavior based on user interactions, preferences, and performance.

The goal of this project is to show how adaptive and intelligent user interfaces can improve user experience using basic AI-related techniques.

## Features

- Theme adaptation (Light/Dark mode)
- Tracks user interaction (most used section)
- Reorders dashboard based on usage
- Adaptive welcome messages
- Quiz-based recommendation system
- Personalized study suggestions

## AI-Based Adaptation

This system uses basic AI logic such as:

- Tracking user behavior (click frequency)
- Storing preferences using local storage
- Rule-based decision making
- Dynamic UI updates based on user actions

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage (for saving user data)

## How to Run

1. Download or clone the repository
2. Open `index.html` in a web browser
3. Start interacting with the dashboard
4. Refresh the page to see adaptive behavior
